# Price Calculator

## Usage
* Clone the repo: 
`git clone https://github.com/rx/price_calculator.git`
* Run the calculator: `./price_calculator`
* Prosper

## Specs
To run the specs

```ruby
bundle install
bundle exec rspec
```
